Hướng dẫn sử dụng

Back end: Folder TonVinhHienMau
Front end:  Folder ton-vinh-hien-mau

Back-end:
+ Chạy lệnh update-database ở phần Package Manager Console
+ Khi chạy back-end chạy cổng localhost:5000
+ Đăng ký acount phía back-end thông qua Swagger với API Register.


Front-end: Cần cài đặt Angular và một số thư viện: Primege, Zoro
+ Khi chạy front-end chạy cổng localhost:4200
+ Có thể chạy bằng npm hoặc yarn.
